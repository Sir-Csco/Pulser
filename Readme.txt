Pulser pings a server to check if server is live or offline.

if server is online :
  Pulser sends an email to the client informing them that the server is online.
  
if server is offline :
  pulser sends an email to the client informing them that the server is offline, and appends
  the email body with an IPstatus.
  
Arguments :
    1. client's email address.
    2. server address.
